# Nonprofit AI Operations Assistant

## Overview

This project demonstrates how Claude can help nonprofit organizations improve volunteer coordination, internal communication, documentation, and day-to-day operations.

Many nonprofits have limited staff and resources. This assistant is designed to reduce repetitive work while helping staff focus on serving their communities.

## Key Features

- Volunteer onboarding support
- AI-generated meeting summaries
- Email drafting assistance
- Frequently asked questions assistant
- Standard Operating Procedures (SOP) generation
- Documentation support

## Target Users

- Nonprofit organizations
- Volunteer coordinators
- Program managers
- Community organizations
